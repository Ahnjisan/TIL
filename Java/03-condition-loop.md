# Java 조건문과 반복문

## 1. 조건문이란?

조건문은 특정 조건에 따라 실행할 코드를 선택하는 문법이다.

프로그램은 항상 위에서 아래로만 실행되는 것이 아니라, 상황에 따라 다른 동작을 해야 할 때가 있다.

예를 들어 로그인 성공 여부, 점수에 따른 등급, 나이에 따른 권한 등을 판단할 수 있다.

    로그인 정보가 맞다
    → 로그인 성공 처리

    로그인 정보가 틀리다
    → 로그인 실패 메시지 출력

이처럼 조건에 따라 실행 흐름을 나누는 문법이 조건문이다.

## 2. if문

`if`문은 가장 기본적인 조건문이다.

조건식이 `true`이면 블록 안의 코드가 실행되고, `false`이면 실행되지 않는다.

기본 형식은 다음과 같다.

    if (조건식) {
        실행할 코드
    }

예시는 다음과 같다.

    int score = 90;

    if (score >= 80) {
        System.out.println("합격입니다.");
    }

위 코드에서 `score >= 80`이 true이므로 `"합격입니다."`가 출력된다.

## 3. if-else문

`if-else`문은 조건이 참일 때와 거짓일 때 실행할 코드를 나누는 문법이다.

기본 형식은 다음과 같다.

    if (조건식) {
        조건이 true일 때 실행
    } else {
        조건이 false일 때 실행
    }

예시는 다음과 같다.

    int score = 70;

    if (score >= 80) {
        System.out.println("합격입니다.");
    } else {
        System.out.println("불합격입니다.");
    }

조건식이 true이면 if 블록이 실행되고, false이면 else 블록이 실행된다.

## 4. if-else if-else문

조건이 여러 개인 경우 `else if`를 사용할 수 있다.

기본 형식은 다음과 같다.

    if (조건식1) {
        조건식1이 true일 때 실행
    } else if (조건식2) {
        조건식2가 true일 때 실행
    } else {
        모든 조건이 false일 때 실행
    }

예시는 다음과 같다.

    int score = 85;

    if (score >= 90) {
        System.out.println("A");
    } else if (score >= 80) {
        System.out.println("B");
    } else if (score >= 70) {
        System.out.println("C");
    } else {
        System.out.println("F");
    }

위 코드에서는 `score`가 85이므로 `"B"`가 출력된다.

## 5. 조건식

조건식은 결과가 `true` 또는 `false`로 나오는 식이다.

조건식에는 비교 연산자와 논리 연산자가 자주 사용된다.

대표적인 비교 연산자는 다음과 같다.

| 연산자 | 의미 |
|---|---|
| `==` | 같다 |
| `!=` | 같지 않다 |
| `>` | 크다 |
| `<` | 작다 |
| `>=` | 크거나 같다 |
| `<=` | 작거나 같다 |

예시는 다음과 같다.

    age >= 20
    score == 100
    name != "admin"

프로그래밍에서 `=`는 값을 할당하는 연산자이고, `==`는 값을 비교하는 연산자이다.

## 6. 논리 연산자

여러 조건을 함께 판단할 때는 논리 연산자를 사용한다.

| 연산자 | 의미 |
|---|---|
| `&&` | 그리고 |
| `||` | 또는 |
| `!` | 부정 |

예시는 다음과 같다.

    int age = 25;
    boolean hasTicket = true;

    if (age >= 20 && hasTicket) {
        System.out.println("입장 가능합니다.");
    }

`&&`는 두 조건이 모두 true일 때 전체 결과가 true가 된다.

`||`는 둘 중 하나라도 true이면 전체 결과가 true가 된다.

`!`는 true를 false로, false를 true로 바꾼다.

## 7. switch문

`switch`문은 하나의 값에 따라 실행할 코드를 나누는 조건문이다.

값이 여러 경우 중 하나로 나뉠 때 사용할 수 있다.

기본 형식은 다음과 같다.

    switch (값) {
        case 값1:
            실행할 코드
            break;
        case 값2:
            실행할 코드
            break;
        default:
            기본 실행 코드
    }

예시는 다음과 같다.

    int menu = 2;

    switch (menu) {
        case 1:
            System.out.println("회원 조회");
            break;
        case 2:
            System.out.println("회원 등록");
            break;
        case 3:
            System.out.println("회원 삭제");
            break;
        default:
            System.out.println("잘못된 메뉴입니다.");
    }

위 코드에서는 `menu` 값이 2이므로 `"회원 등록"`이 출력된다.

## 8. break의 역할

`switch`문에서 `break`는 해당 case 실행을 끝내고 switch문을 빠져나가는 역할을 한다.

`break`를 작성하지 않으면 다음 case까지 이어서 실행될 수 있다.

따라서 특별한 의도가 없다면 각 case 끝에 `break`를 작성하는 것이 안전하다.

## 9. 반복문이란?

반복문은 같은 작업을 여러 번 실행할 때 사용하는 문법이다.

예를 들어 1부터 10까지 출력하거나, 배열에 있는 값을 하나씩 확인하거나, 특정 조건을 만족할 때까지 작업을 반복할 수 있다.

반복문을 사용하지 않으면 같은 코드를 여러 번 작성해야 한다.

    System.out.println(1);
    System.out.println(2);
    System.out.println(3);

반복문을 사용하면 위와 같은 반복 작업을 더 효율적으로 작성할 수 있다.

## 10. for문

`for`문은 반복 횟수가 정해져 있을 때 많이 사용한다.

기본 형식은 다음과 같다.

    for (초기식; 조건식; 증감식) {
        반복 실행할 코드
    }

예시는 다음과 같다.

    for (int i = 1; i <= 5; i++) {
        System.out.println(i);
    }

위 코드는 1부터 5까지 출력한다.

for문의 흐름은 다음과 같다.

    초기식 실행
    → 조건식 확인
    → 조건식이 true이면 블록 실행
    → 증감식 실행
    → 다시 조건식 확인
    → 조건식이 false이면 반복 종료

## 11. for문의 구성 요소

for문은 초기식, 조건식, 증감식으로 구성된다.

| 구성 요소 | 설명 |
|---|---|
| 초기식 | 반복에 사용할 변수 초기화 |
| 조건식 | 반복을 계속할 조건 |
| 증감식 | 반복 변수 값 변경 |

예시는 다음과 같다.

    for (int i = 0; i < 10; i++) {
        System.out.println(i);
    }

여기서 `int i = 0`은 초기식이다.

`i < 10`은 조건식이다.

`i++`는 증감식이다.

## 12. while문

`while`문은 조건이 true인 동안 반복을 계속하는 문법이다.

기본 형식은 다음과 같다.

    while (조건식) {
        반복 실행할 코드
    }

예시는 다음과 같다.

    int count = 1;

    while (count <= 5) {
        System.out.println(count);
        count++;
    }

`while`문은 반복 횟수가 명확하지 않고, 특정 조건이 만족되는 동안 반복해야 할 때 사용할 수 있다.

## 13. do-while문

`do-while`문은 먼저 코드를 한 번 실행한 뒤 조건을 확인한다.

기본 형식은 다음과 같다.

    do {
        반복 실행할 코드
    } while (조건식);

예시는 다음과 같다.

    int count = 1;

    do {
        System.out.println(count);
        count++;
    } while (count <= 5);

`do-while`문은 조건이 처음부터 false여도 최소 한 번은 실행된다.

## 14. for문과 while문의 차이

`for`문과 `while`문은 모두 반복문이지만 주로 사용하는 상황이 다르다.

| 구분 | 주 사용 상황 |
|---|---|
| for문 | 반복 횟수가 명확할 때 |
| while문 | 조건 중심으로 반복할 때 |
| do-while문 | 최소 한 번은 실행해야 할 때 |

예를 들어 1부터 10까지 출력하는 경우에는 for문이 적합하다.

사용자가 올바른 값을 입력할 때까지 반복하는 경우에는 while문이 적합할 수 있다.

## 15. break

`break`는 반복문을 즉시 종료할 때 사용한다.

예시는 다음과 같다.

    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break;
        }
        System.out.println(i);
    }

위 코드는 i가 5가 되는 순간 반복문을 종료한다.

따라서 1부터 4까지만 출력된다.

## 16. continue

`continue`는 현재 반복을 건너뛰고 다음 반복으로 넘어갈 때 사용한다.

예시는 다음과 같다.

    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue;
        }
        System.out.println(i);
    }

위 코드는 i가 3일 때 출력하지 않고 다음 반복으로 넘어간다.

따라서 1, 2, 4, 5가 출력된다.

## 17. 중첩 반복문

반복문 안에 또 다른 반복문을 작성할 수 있다.

이를 중첩 반복문이라고 한다.

예시는 다음과 같다.

    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 2; j++) {
            System.out.println("i=" + i + ", j=" + j);
        }
    }

중첩 반복문은 표 형태의 데이터, 2차원 배열, 구구단 등을 처리할 때 자주 사용된다.

## 18. 조건문과 반복문의 관계

조건문과 반복문은 프로그램의 실행 흐름을 제어하는 문법이다.

| 문법 | 역할 |
|---|---|
| 조건문 | 조건에 따라 실행 흐름을 나눔 |
| 반복문 | 같은 코드를 여러 번 실행함 |

조건문은 “어떤 상황에서 실행할 것인가”를 판단한다.

반복문은 “몇 번 또는 언제까지 실행할 것인가”를 결정한다.

두 문법은 함께 사용되는 경우가 많다.

예를 들어 반복문으로 숫자를 하나씩 확인하면서, 조건문으로 짝수인지 홀수인지 판단할 수 있다.

## 19. 정리

이번 학습에서는 Java의 조건문과 반복문을 정리했다.

핵심 개념은 다음과 같다.

- 조건문은 조건에 따라 실행할 코드를 선택하는 문법이다.
- `if`문은 조건식이 true일 때 실행된다.
- `if-else`문은 true와 false 상황을 나누어 처리한다.
- `else if`는 여러 조건을 순차적으로 검사할 때 사용한다.
- `switch`문은 하나의 값에 따라 실행 흐름을 나눌 때 사용한다.
- `break`는 switch문이나 반복문을 종료할 때 사용한다.
- 반복문은 같은 코드를 여러 번 실행할 때 사용한다.
- `for`문은 반복 횟수가 명확할 때 사용한다.
- `while`문은 조건이 true인 동안 반복한다.
- `do-while`문은 최소 한 번 실행한 뒤 조건을 검사한다.
- `continue`는 현재 반복을 건너뛰고 다음 반복으로 넘어간다.
- 조건문과 반복문은 프로그램의 실행 흐름을 제어하는 핵심 문법이다.
